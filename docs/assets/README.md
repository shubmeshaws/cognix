# Documentation media

Add screenshots and short GIFs here for the setup guide and README.

Suggested captures (record from a running local or staging install):

| File | What to capture |
|------|-----------------|
| `dashboard-overview.png` | Overview page with pod health and agent terminal |
| `meshy-chat.gif` | Meshy answering a cluster question (10–15 s) |
| `setup-healthchecks.png` | Setup page with all checks green |
| `settings-llm.png` | Settings → Agent with LLM provider configured |
| `connect-cluster.png` | Clusters page connecting a kubeconfig |

After adding files, they appear automatically in [SETUP.md](../SETUP.md) and the root README.

**Tip:** On macOS use **Cmd+Shift+5** for screen recording; export as GIF with [gifski](https://gif.ski/) or an online converter.
